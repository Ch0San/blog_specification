요약

총 13개 테이블, Spring Security(폼 로그인), Thymeleaf UI, Kakao 지도(게시글) 통합.

Java 17 Spring Boot 3 MySQL 8 Thymeleaf JPA + MyBatis

원문: PROJECT_SPECIFICATION.md

기술 스택

WAS/DB/Front

WAS

Apache Tomcat(임베디드), 포트 8080

DB

MySQL 8.0 (utf8mb4), HikariCP, H2(테스트)

보안

Spring Security(Form Login, Remember-Me, URL 인가)

Persistence

JPA + MyBatis 혼용 (학습 목적), Hibernate SQL 로그

프론트

Thymeleaf + Vanilla JS, 스타일 가벼운 네온 포인트

외부 API

Kakao Maps JavaScript API (게시글 위치)
