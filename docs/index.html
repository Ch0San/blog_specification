블로그 시스템 명세
Spring Boot + MySQL

게시글/댓글/좋아요, 동영상 스토리, 공지사항까지 포함한
학습용 블로그 프로젝트의 핵심 스펙을 한 눈에 정리했습니다.

빠른 실행 안내 데이터베이스 구조

요약

총 13개 테이블, Spring Security(폼 로그인), Thymeleaf UI, Kakao 지도(게시글) 통합.

Java 17 Spring Boot 3 MySQL 8 Thymeleaf JPA + MyBatis

기술 스택

WAS/DB/Front

WAS

Apache Tomcat(임베디드), 포트 8080

DB

MySQL 8.0.42, HikariCP, H2

보안

Spring Security

Persistence

JPA + MyBatis, Hibernate SQL 로그

프론트

Thymeleaf + Vanilla JS

외부 API

Kakao Maps JavaScript API

데이터베이스 개요

13 Tables

핵심 테이블

테이블설명
